קונספט ארכיטקטורת מוצר AI

מקורות נתונים

הנתונים מתקבלים ממגוון מקורות:

  • ווצאפ - חיבור API שמקבל הודעות.
  • סלאק - חיבור API לאיסוף נתונים.
  • אתר - ממשק משתמש לאיסוף נתונים.
  • אפליקציה - חיבור API לאיסוף נתונים.

מודל בינה מלאכותית

המודל עובר עיבוד נתונים ומבצע:

  • עיבוד נתונים מהמקורות השונים.
  • שימוש באלגוריתמים של למידת מכונה.
  • עיבוד שפה טבעית.

מאגרי נתונים

נתוני המודל נשמרים במאגרי נתונים:

  • מסד נתונים לאחסון נתונים (SQL/NoSQL).
  • פונקציות מותאמות אישית לעיבוד וניתוח הנתונים.
