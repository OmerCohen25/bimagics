Technology & Architecture

Built on a modern, scalable, and secure stack using industry-leading tools.

Tech Stack

Google Gemini, Node.js, React, Google Cloud Run, Firebase (Auth, Firestore, Storage), Meta API, GitHub Actions CI/CD.

Serverless & Scalable

The entire backend is built on serverless technology, meaning it scales automatically from ten users to ten million without manual intervention.

Secure by Design

We use Firebase's robust security rules and authentication to protect user data. The AI generates configuration data, not executable code, preventing malicious injections.
