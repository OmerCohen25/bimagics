לאוטונומי וחכם

WIZBI היא לא עוד מערכת דוחות. היא פלטפורמת Data + AI שמחברת את המלאי, הפיננסים (Morning), הלקוחות והתפעול למקום אחד, ומפעילה סוכנים חכמים (Agents) שמנהלים את העסק בשבילך.

ראה איך זה עובד איך זה בנוי? (Tech Stack)
היי וויזבי, מה המצב עם המלאי והמכירות השבוע? יש משהו דחוף?

סיכום מצב בזמן אמת:

  • מכירות השבוע: ₪42,500 (עלייה של 12% משבוע שעבר).
  • מלאי: מוצר "אייפון 17" ירד מתחת ל-5 יחידות.
  • תפעול: זוהתה ירידה בתפוקה באתר סולארי "נגב".

פלטפורמה אחת. אינסוף אפשרויות.

המודל הגאוני של WIZBI: תשתית ליבה חזקה אחת (Core), שעליה מתלבשים מודולים (Packs) מותאמים לכל סוג עסק.

למה זה גאוני תפעולית? (Two-Man Ops)

בנינו את המערכת כך שצוות של שני יזמים יכול לנהל אלפי לקוחות. בלי שרתים לנהל, בלי פיתוחים כפולים לכל לקוח. Architecture as a Strategy.

Cloud Run

שרתים שרצים רק שיש בקשות (Serverless). עלות אפס כשהמערכת נחה, וסקייל אינסופי בעומס.

Multi-Tenancy

אותו קוד משרת את כולם. הפרדה לוגית מלאה של נתונים, אבל תחזוקה של מערכת אחת בלבד.

BigQuery + Firestore

שילוב של דאטה "חם" לתפעול שוטף ודאטה "קר" לאנליטיקה כבדה ו-AI בעלויות מצחיקות.

CI/CD Factory

אוטומציה מלאה משורת הקוד ועד לפרודקשן. פריסת גרסאות ופיצ'רים חדשים לכל הלקוחות בלחיצה.

GCP Full Stack Architecture
Firebase Hosting (UI) → Cloud Run (API Gateway) → Vertex AI (Agents) ↔ BigQuery (DWH)

אנחנו בונים את המוח של העסק הבא.

רוב העסקים היום יושבים על מכרה זהב של נתונים, אבל מנהלים אותו כמו בעידן האבן. הם צמאים לאוטומציה, אבל אין להם תקציב לפרויקטי ענק.

WIZBI באה לשנות את זה. אנחנו מנגישים את הטכנולוגיה הכי מתקדמת בעולם (Generative AI) בצורה הכי פשוטה שיש: חיבור אחד, וזה עובד.
