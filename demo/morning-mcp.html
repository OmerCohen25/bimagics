איך זה עובד מאחורי הקלעים?

  • WhatsApp – הלקוח שולח הודעה רגילה למספר ייעודי.
  • Claude/LLM – מנתח את הבקשה בשפה טבעית, מבין את הכוונה ומחלץ את הפרמטרים (שם לקוח, סכום).
  • MCP Server (Cloud Run) – שרת מאובטח שרץ בענן עם סקיילינג אוטומטי. הוא מקבל את הפקודה מהמודל ומפעיל את הכלים המתאימים.
  • Tools (Functions) – השרת משתמש בפונקציות ייעודיות:
    • `find_or_create_client` – מאתר או יוצר לקוח חדש במערכת חשבונית ירוקה.
    • `create_invoice_receipt` – מפיק מסמך TAX_INVOICE_RECEIPT ושולח אותו ללקוח.
  • API Integration – כל התקשורת עם חשבונית ירוקה מתבצעת דרך ה-API הרשמי שלהם.

כל התהליך מתבצע בזמן אמת – המשתמש רק מדבר בשפה טבעית, והשירות מטפל בחשבוניות, בלקוחות ובמסמכים באופן אוטומטי 🚀
