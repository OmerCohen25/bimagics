The Ecosystem

Two Products. One Infinite Loop.

Wizbi isn't just an agent framework. It's a factory that builds agents, and agents that command the factory.

Wizbi Control Plane

The GCP Project Factory

An agnostic backend that provisions entire Google Cloud projects in seconds. It sets up Cloud Run, Firestore, Secret Manager, and CI/CD pipelines. Every project it creates comes with built-in MCP APIs out-of-the-box.

  • Zero-to-Prod in 1 Click
  • Automated Billing Export setup
  • Strict IAM & Secret isolation

Wizbi Agents

The Autonomous Swarm

The ultimate AI assistant template. Connects Gemini 3.1 to WhatsApp, Telegram, and Web. Armed with MCP skills like Google Workspace, Monday.com, and local schedulers, capable of executing complex business logic natively.

  • Proactive cron-based heartbeats
  • Long-term Firestore memory
  • Multi-channel webhook routing

An entire DevOps team, in a single prompt.

FinOps AI (Self-Aware Billing)

Agents know how much they cost. Native BigQuery integration allows your agent to query its own Vertex AI and Cloud Run usage, alerting you or pausing non-critical tasks before hitting budget caps.

Fractal Architecture

Self-replicating by design. The Master Agent can command the Control Plane via MCP to provision entirely new GCP projects and deploy fresh agent instances for specific departments or clients.

100% MCP Native

No proprietary plugin lock-in. The ecosystem is built entirely around the Model Context Protocol. Wrote a custom MCP for your internal database? It works out-of-the-box across all your Wizbi agents.

Scale-to-Zero Cloud

Pure Serverless infrastructure on Cloud Run and Firebase. When there are no messages, you pay $0. When traffic spikes, it instantly scales to thousands of containers.

The Obvious Comparison

Wizbi vs. Standard Frameworks

Why code frameworks are nice, but Cloud Operating Systems are the future.

OpenClaw & Others

  • Code Only: You still have to figure out hosting, DB provisioning, secrets management, and CI/CD.
  • Financial Blindness: Agents run blindly. A prompt injection or infinite loop could cost you thousands in LLM API bills.
  • Always-On Servers: Typically requires an EC2 or VPS running 24/7 just to listen for webhooks.
  • Agent Spaghetti: Running multiple agents often means cramming them into one server, breaking security isolation.
THE WIZBI WAY

Wizbi OS

  • Infrastructure in a Box: Automated GCP deployment provisions a secure, production-ready cloud in seconds.
  • Built-in FinOps: The agent queries the Billing API to check its own daily cost and halts non-critical tasks if nearing budget.
  • Scale to Zero: Cost-effective webhook architecture. Cloud Run containers sleep when idle and wake instantly.
  • Tenant Isolation: Agent A can deploy Agent B into a completely separate GCP project with its own DB and keys.
